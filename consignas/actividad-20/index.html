Funciones de
Conteo

Investigá, entendé y demostrá el uso de las funciones que permiten contar datos en una hoja de cálculo, cada una con su propio criterio y lógica.

CONTAR() CONTARA() CONTAR.BLANCO() CONTAR.SI() CONTAR.SI.CONJUNTO() Investigación Ejemplo propio

Objetivo de aprendizaje

Comprender qué tipo de datos considera cada función de conteo, cuándo conviene usar una u otra, y ser capaz de diseñar ejemplos propios que demuestren su funcionamiento correcto en situaciones concretas.

Título de la planilla

La primera celda del archivo debe llevar el siguiente título exacto, con formato de encabezado destacado:

" Investigación sobre funciones para contar "

Funciones a investigar

Para cada una de las cinco funciones tenés que desarrollar los mismos cinco puntos. El orden y la estructura deben repetirse para facilitar la lectura.

CONTAR() Cuenta celdas con números
CONTARA() Cuenta celdas no vacías
CONTAR.BLANCO() Cuenta celdas vacías
CONTAR.SI() Cuenta con una condición
CONTAR.SI.CONJUNTO() Cuenta con múltiples condiciones

Los cinco puntos por función

Este esquema se repite para las cinco funciones. Usá una hoja separada o una sección bien diferenciada para cada una.

1

Nombre de la función

Escribí el nombre exacto de la función tal como se escribe en la hoja de cálculo, incluyendo los paréntesis.

Para tener en cuenta

El nombre puede variar levemente según el software (Google Sheets, LibreOffice, Excel). Si el que usás tiene un nombre diferente, aclaralo.

2

Descripción

Explicá con tus propias palabras respondiendo las tres preguntas:

  • ¿Para qué sirve esta función?
  • ¿Qué tipo de datos tiene en cuenta y cuáles ignora?
  • ¿En qué situaciones reales la usarías?
Para reflexionar

No copies la definición tal cual la encontrás en internet. Reformulala: si no podés explicarla con tus palabras, probablemente todavía no la entendiste del todo.

3

Sintaxis y parámetros

Escribí la estructura completa de la función y explicá cada parte:

  • ¿Cuántos parámetros tiene? ¿Cuáles son obligatorios y cuáles opcionales?
  • ¿Qué tipo de dato espera cada parámetro: un rango, un número, un texto, una condición?
  • ¿Qué devuelve si el rango está vacío o no hay coincidencias?
Para reflexionar

Para CONTAR.SI() y CONTAR.SI.CONJUNTO(): ¿cómo se escribe un criterio que compare un número? ¿Y uno que busque un texto exacto? ¿Y uno con comodines?

4

Ejemplo práctico Obligatorio

Diseñá una tabla de datos propia (no copies ejemplos de internet) y aplicá la función para resolver una situación concreta. El tema de la tabla es libre.

  • La tabla debe tener al menos 6 filas de datos (sin encabezado).
  • La fórmula debe estar activa en la planilla, no solo escrita como texto.
  • Indicá claramente qué celda contiene la función y cuál es el resultado obtenido.
Para reflexionar

Pensá en una tabla que tenga celdas vacías intencionalmente, valores numéricos y valores de texto mezclados. Eso va a permitirte mostrar mejor las diferencias entre las funciones.

5

Análisis del resultado

Explicá brevemente por qué el número que devuelve la función es correcto. Contá cuántos ítems cumplen la condición y dónde están en tu tabla.

Para reflexionar

Este punto es para demostrar que entendés lo que pasó. No alcanza con decir "el resultado es 3". Explicá cuáles son esas 3 celdas y por qué las demás no se contaron.

Requisitos generales

01
Redacción propia

No copiar definiciones textuales. Las descripciones deben estar escritas con las propias palabras del estudiante.

02
Fórmulas activas

03
Claridad y orden

04
Análisis incluido

Para profundizar Opcional

Si terminaste el trabajo principal y querés ir más lejos:

Comparar CONTAR y CONTARA

CONTAR.SI.CONJUNTO con dos condiciones

Situación real
