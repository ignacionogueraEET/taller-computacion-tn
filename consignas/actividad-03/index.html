Inventario Técnico del Equipo de Trabajo

El coordinador del taller necesita actualizar el inventario técnico de las máquinas del laboratorio. Tu misión es identificar las especificaciones de hardware y software de la computadora que estás utilizando en este momento y generar un informe formal de una página, utilizando el procesador de textos del sistema.

El documento final debe ser limpio, profesional y accesible, cumpliendo con los estándares de legibilidad y contraste para cualquier usuario, incluyendo personas con baja visión o daltonismo.

Pasos a seguir

  1. Obtención de datos del sistema

    En tu computadora con Debian Cinnamon, hacé clic en el Menú (esquina inferior izquierda) > Administración > Información del sistema. Anotá los siguientes datos:

    • Sistema Operativo (ej: Debian GNU/Linux)
    • Entorno de escritorio (ej: Cinnamon)
    • Procesador (Modelo y velocidad)
    • Memoria (RAM)
    • Almacenamiento (Disco y capacidad)
    • Gráficos (Tarjeta de video)

    Opcional para perfiles avanzados: Podés usar la terminal y escribir inxi -F o neofetch para obtener los mismos datos de forma más detallada.

  2. Crear el documento en LibreOffice Writer

    Abrí LibreOffice Writer (nativo en Debian) o el procesador de textos que prefieras. Configurá la página con los siguientes parámetros de accesibilidad:

    • Título: "Informe Técnico del Equipo de Trabajo" (Fuente Arial 14, Negrita, Centrado).
    • Cuerpo del texto: Fuente Arial o similar, tamaño 11, interlineado 1.5.
    • Encabezado de página: Tu Nombre y Curso.
    • Pie de página: Número de página.

  3. Diseñar la tabla de especificaciones

    Insertá una tabla de 2 columnas ("Componente" y "Especificación") con tantas filas como datos hayas recopilado. Aplicá el siguiente formato obligatorio:

    • Encabezados de tabla: Color de fondo oscuro (ej: Azul oscuro #003366 o Gris oscuro #333333) con texto en Blanco (#FFFFFF) para garantizar un contraste superior a 4.5:1.
    • Bordes: La tabla debe tener bordes visibles en todas las celdas. No uses solo el color para diferenciar los encabezados; el borde es obligatorio para usuarios con daltonismo o impresión en blanco y negro.

    Validá el contraste de los colores elegidos con la herramienta WebAIM Contrast Checker(abre en nueva pestaña) antes de aplicarlos.

  4. Guardar y exportar

    Guardá el archivo como: InventarioPC_Apellido_Nombre.odt (formato nativo de Software Libre) o .docx si necesitás compatibilidad con otros sistemas. Verificá que la tabla se vea correctamente al abrir el documento en modo de solo lectura.

Pensamiento Computacional

Antes de empezar, reflexioná sobre estos tres pilares:

  • Descomposición: Obtener datos del sistema → Organizarlos en una tabla → Aplicar formato profesional y accesible.
  • Abstracción: Solo nos interesan los datos técnicos relevantes (CPU, RAM, SO), no detalles anecdóticos.
  • Algoritmo: 1) Abrir "Información del sistema" → 2) Anotar datos clave → 3) Abrir LibreOffice Writer → 4) Crear tabla y aplicar formato WCAG → 5) Guardar y exportar.

Cómo entregar tu trabajo

Subí el archivo del documento con el nombre InventarioPC_Apellido_Nombre.odt (o .docx) a la plataforma indicada por el docente. Asegurate de que:

  • La tabla tenga bordes visibles en todas las celdas.
  • El contraste entre el fondo de los encabezados y el texto sea ≥ 4.5:1.
  • El documento no supere una página.
  • El encabezado contenga tu nombre y curso, y el pie de página el número de página.

Fecha de entrega: 7 días a partir de la publicación de la consigna.

Contexto

Objetivo

Identificar las especificaciones técnicas de hardware y software de la computadora en uso y generar un informe formal accesible en procesador de textos, aplicando criterios WCAG 2.1 Nivel AA.

Herramientas

  • Debian GNU/Linux + Cinnamon
  • LibreOffice Writer
  • WebAIM Contrast Checker
  • Terminal (opcional: inxi -F / neofetch)

Modalidad

Individual. Duración estimada: 2 horas de clase (obtención de datos + diseño del documento + validación de accesibilidad).
