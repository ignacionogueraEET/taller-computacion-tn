Actividad 7: Referencia absoluta y relativa – Copia de fórmula

📘 Consigna

Crear un panel de control donde se coloque el número correspondiente a la tabla que se desea generar. Debajo se debe mostrar la tabla del número configurado. Las columnas serán: Multiplicando, Multiplicador, Producto.

Se deberá crear las fórmulas correspondientes y copiarlas para cubrir toda la tabla del 0 al 10. Esta tabla debe actualizarse automáticamente al cambiar el número en el panel de control. Las fórmulas deben poder ser copiadas para acelerar el trabajo (usando referencia absoluta y relativa).

Aplicar formato a las celdas: color de fondo, color de letra, bordes.

🎛️ Panel de control

Ingresá un número del 0 al 10. La tabla se actualizará automáticamente.
