# 📊 Atajos de Teclado para Google Sheets

Material educativo interactivo para estudiantes de computación que se inician en el uso de Google Sheets.

## 🎯 Objetivo

Proporcionar una guía visual, profesional y motivadora de los atajos de teclado más útiles para mejorar la productividad en Google Sheets.

## 🚀 Cómo usar

Este material está diseñado para ser:
- **Visual**: Cada atajo se presenta en tarjetas interactivas con efectos hover
- **Organizado**: Categorizado por funcionalidad (navegación, edición, formato, fórmulas, gestión)
- **Responsive**: Adaptable a dispositivos móviles y tablets
- **Accesible**: Con enlaces a recursos oficiales de Google

## 📚 Contenido

### Atajos incluidos:
- **Navegación y Selección** (6 atajos)
- **Edición de Celdas** (8 atajos)
- **Formato Básico** (6 atajos)
- **Fórmulas y Funciones** (4 atajos)
- **Gestión de Hojas** (3 atajos)

### Recursos externos vinculados:
- Soporte oficial de Google
- Hoja de referencia (Cheat Sheet)
- Guía específica para Mac

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 con variables CSS
- Diseño responsive (Mobile-first)
- Animaciones CSS
- Google Fonts (Inter)

## 📖 Para el aula

Este material puede ser:
1. Alojado en GitHub Pages
2. Compartido mediante enlace en Google Classroom
3. Proyectado en clase como referencia visual
4. Consultado por estudiantes desde cualquier dispositivo

## 📝 Nota para docentes

El diseño incluye:
- Paleta de colores profesional (verde Google Sheets)
- Tipografía legible (Inter)
- Animaciones sutiles que no distraen
- Énfasis visual en atajos más importantes
- Consejo motivador en la introducción
