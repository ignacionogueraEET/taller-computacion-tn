Atajos de Teclado para Google Sheets

Guía práctica para estudiantes de computación

Nivel: Introducción

💡 Consejo profesional: Dominar los atajos de teclado no solo acelera tu trabajo, sino que te permite mantener el flujo de concentración sin interrupciones. Dedica 5 minutos diarios a practicar un nuevo atajo y en pocas semanas notarás la diferencia en tu productividad.

🧭

Navegación y Selección

Ctrl + Inicio
Ir a la celda A1
Ctrl + Fin
Ir a la última celda con datos
Ctrl + Barra espaciadora
Seleccionar columna completa
Ctrl + Shift + Barra espaciadora
Seleccionar fila completa
Ctrl + A
Seleccionar todas las celdas
Ctrl + Shift + Flechas
Seleccionar rango de celdas
✏️

Edición de Celdas

F2
Editar celda activa
Enter
Confirmar y mover abajo
Tab
Confirmar y mover derecha
Esc
Cancelar edición
Ctrl + D
Rellenar hacia abajo
Ctrl + R
Rellenar hacia la derecha
Ctrl + ;
Insertar fecha actual
Ctrl + Shift + ;
Insertar hora actual
🎨

Formato Básico

Ctrl + B
Negrita
Ctrl + I
Cursiva
Ctrl + U
Subrayado
Ctrl + Shift + $
Formato moneda
Ctrl + Shift + %
Formato porcentaje
Ctrl + Shift + #
Formato fecha
🔢

Fórmulas y Funciones

Ctrl + =
Insertar suma automática (Σ)
F4
Alternar referencias absolutas/relativas
Ctrl + Shift + Enter
Insertar fórmula matricial
Ctrl + '
Copiar fórmula de celda superior
📑

Gestión de Hojas

Shift + F11
Insertar nueva hoja
Ctrl + Shift + Page Up/Down
Mover entre hojas
Alt + Shift + K
Eliminar hoja actual

🍎 ¿Usas Mac?

Reemplaza Ctrl por ⌘ Command y Alt por ⌥ Option en todos los atajos anteriores.
